# Dashboard de Acompanhamento de Pacientes

Este é um dashboard moderno para nutricionistas e profissionais de saúde acompanharem o progresso de seus pacientes.

## Funcionalidades

- **Dashboard Geral**: Visão geral de pacientes, perda de peso e tarefas.
- **Gestão de Pacientes**: Lista de pacientes com busca e status.
- **Detalhes do Paciente**: Histórico de peso (gráficos), medidas, suplementos e dicas.
- **Tarefas**: Sistema de tarefas pendentes e concluídas.

## Como Rodar o Projeto

1.  **Instalar Dependências** (se ainda não fez):
    ```bash
    npm install
    ```

2.  **Rodar o Servidor de Desenvolvimento**:
    ```bash
    npm run dev
    ```

3.  **Acessar no Navegador**:
    Abra o link que aparecerá no terminal (geralmente `http://localhost:5173`).

## Tecnologias

- React + Vite
- Tailwind CSS
- Recharts (Gráficos)
- Lucide React (Ícones)
